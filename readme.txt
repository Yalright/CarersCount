== Description ==
Carers Count is a custom WordPress theme designed specifically for the Carers Count network. This theme provides a responsive, user-friendly interface tailored to meet the unique needs of carers and their networks. It offers modern design features and compatibility with popular WordPress plugins.

== Installation ==
1. Download the theme ZIP file from [https://bitbucket.org/mikedistras/barnsleycarers](https://bitbucket.org/mikedistras/barnsleycarers).
2. Go to your WordPress admin dashboard.
3. Navigate to Appearance > Themes > Add New > Upload Theme.
4. Upload the ZIP file, click "Install Now," and activate the theme.
